# 前端常用插件及框架

- 插件一般是为解决某一问题而存在的，功能单一
- 框架提供了完整的解决方案，框架的控制权在框架本身，使用者需按照框架的规范开发，既能开发移动端也能开发pc端

## 前端常用插件

- 轮播图插件网站：[swiper](https://www.swiper.com.cn/)
- superslide
- iscroll
- zy.media.js 视频插件

## 前端常用框架

- bootstrap
- Vue
- Angular
- React
