# 移动端滚动条动态水平居中

- **关键代码**

```js
//滚动条位置居中,使得标题能在移动端被居中
    function middleScroll(step,totalTime){
      let f = document.getElementById("BOX_outside")
      let s = document.getElementById("BOX_inside")
      let delayTime= totalTime / ((s.clientWidth - f.clientWidth) / 2 / step)

      let scrollTimer = setInterval(
        () => {
          if (f.scrollLeft <= (s.clientWidth - f.clientWidth) / 2 )
          {
            f.scrollLeft+=step;
          }else
          {
            clearInterval(scrollTimer);
          }
      }, delayTime);
  }
  window.addEventListener('loaded',middleScroll(2,500)) //移动端动态局居中效果，每次移动2px 在500ms内使标题居中
```
