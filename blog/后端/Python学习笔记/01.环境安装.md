## miniconda3环境安装

**安装**

```bash
# 1.解压安装miniConda
# 2.然后在bash中执行：
$ /d/app/miniconda3/Scripts/conda init bash
$ /d/app/miniconda3/Scripts/conda init powershell
$ /d/app/miniconda3/Scripts/conda init cmd.exe
```

**创建环境**

```bash
conda create --name learn-python -y python=3.9
```

**列出环境**

```bash
conda env list
```

**激活环境**

```bash
conda activate learn-python
```

**解除激活环境**

```bash
conda deactivate
```
