# 第4章_2.二叉树的概念和实现
* 二叉树（binary tree）
* 每个节点最多拥有两个子节点
* 有N个节点的二叉树平均深度为O(√N)
* 二叉查找树的平均深度为O(log N)
* 画图时，用圆圈和直线表示，因为二叉树实际上是图（graph）
* 最差情况下其深度为N-1
![](/《数据结构与算法分析》学习笔记/attachments/Clipboard_2022-05-28-04-00-27.png)

## 具体实现
* **构成：** 一个值域两个指针域
```c
#define ElementType int
typedef struct TreeNode * ptrToNode;
typedef struct ptrToNode * Tree;
struct TreeNode
{
  ElementType Element;
  ptrToNode Left;
  ptrToNode Right;
};
```
